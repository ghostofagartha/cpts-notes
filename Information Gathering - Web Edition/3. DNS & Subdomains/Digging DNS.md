**Dig Command**
- `dig [domain name]`
- `dig [domain name] [record type]`
- `dig [domain name] ANY`
- `dig @[name server] [domain name]`
- `dig +trace [domain name]`: Show full path of DNS resolution
- `dig -x [ip address]`: Reverse lookup
- `dig +short [domain name]`: Concise
- `dig +noall +answer [domain name]`: Display only answer section of the query
