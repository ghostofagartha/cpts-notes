- A text file in the root directory of a website
- Acts as a virtual "etiquette guide" for bots

**How robots.txt Works**
- ```
  User-agent: *
  Disallow: /private/
  ```
- Prevents all the bots from accessing directory `/private` and its sub directories and file

**robots.txt Structure**
1. **User-agent:** Specifies which crawler or bot this rule applies to
2. **Directives:** The specific instruction to the user-agent
	- Common directives include:
		1. `Disallow`: Prevents access
		2. `Allow`: Provides access to the user-agent. Overrides the `Disallow`
		3. `Crawl-delay`: Sets a delay (in seconds) between successive requests
		4. `Sitemap`: Provides URL to an XML sitemap for more efficient crawling

**Why Respect robots.txt**
- Avoiding Overburdening Servers
- Protecting Sensitive Information
- Legal and Ethical Compliance

**robots.txt in Web Reconnaissance**
- Uncovering Hidden Directories
- Mapping Website Structure
- Detecting Crawler Traps
