- Search Engine Discovery is also called OSINT (Open Source Intelligence)

**Why Search Engine Discovery Matters?**
- Open Source
- Breadth of Information
- Ease of Use
- Cost-Effective
- Information pulled can be used for
	- Security Assessment
	- Competitive Intelligence
	- Investigative Journalism
	- Threat Intelligence

**Search Operators**
- `site:`
- `inurl:`
- `filetype:`
- `intitle:`
- `intext:` or `inbody:`
- `cache:`: Displays cached version of a webpage if available
- `link:`: Find pages that link to a specific webpage
- `related:`: Find websites related to a specific webpage
- `info:`: Provides a summary of information about a webpage
- `define:`: Provides definition of a word or phrase
- `numrange:`
- `allintext:`: Find pages containing all specified words in the URL
- `allinurl:`
- `AND`
- `OR`
- `NOT`
- `*` (wildcard)
- `" "`: Searches for exact phrases
- `..` (range search)
- `-`: Excludes terms from the search results
