- **Secure Sockets Layer/Transport Layer Security (SSL/TLS):**
	- A protocol which encrypts communication between your browser and a website
	- Has a **digital certificate**: a small file that verifies a website's identity and allows for secure, encrypted communication

**What are Certificate Transparency Logs**
- `Certificate Transparency (CT) logs are public, append-only ledgers that record instance of SSL/TLS certificates`
- Are like `global registry of certificates`
- Serves following purposes:
	- **Early Detection of Rogue Certificates**
	- **Accountability for Certificate Authorities**
	- **Strengthening the Web PKI (Public Key Infrastructure)**

**Searching of CT Logs**
- Tools
	- `crt.sh`
	- `Censys`
- **crt.sh lookup:**
	- `curl -s "https://crt.sh/?q=facebook&output=json" | jq -r '.[] | select(.name_value | contains("dev")) | .name_value' | sort -u`
	- `curl -s "https://crtsh/?q=facebook.com&output=json" | jq -r '.[] | .name_value' | grep -E '(^|\.)dev\.' | sort -u`
