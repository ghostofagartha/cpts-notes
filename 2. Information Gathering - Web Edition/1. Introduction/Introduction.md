**Primary Goals**
- Identifying Assets
- Discovering Hidden Information
- Analyzing the Attack Service
- Gathering Intelligence

***Types of Reconnaissance***
1. **Active**
	- Directly interacts with the target system
	1. Port Scanning: **High Risk**
	2. Vulnerability Scanning: **High Risk**
	3. Network Mapping: **Medium to High Risk**
	4. Banner Grabbing: **Low Risk**
	5. OS Fingerprinting: **Low Risk**
	6. Service Enumeration: **Low Risk**
	7. Web Spidering: **Low to Medium Risk**
2. **Passive**
	- Without directly interacting
	1. Search Engine Queries: **Very Low Risk**
	2. WHOIS Lookups: **Very Low Risk**
	3. DNS: **Very Low Risk**
	4. Web Archive Analysis: **Very Low Risk**
	5. Social Media Analysis: **Very Low Risk**
	6. Code Repositories: **Very Low Risk**
