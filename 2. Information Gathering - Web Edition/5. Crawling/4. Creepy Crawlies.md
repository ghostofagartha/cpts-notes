**ReconSpider**
- `  wget -O ReconSpider.zip https://academy.hackthebox.com/storage/modules/144/ReconSpider.v1.2.zip`
- `python3 ReconSpider.py [domain]`
- Will create a **results.json** file
	- Contains following JSON keys
		1. **emails**
		2. **links**
		3. **external_files:** URLs of external files
		4. **js_files**: URLs of JavaScript files used by the website
		5. **form_fields:** Form fields found on the domain
		6. **images**
		7. **videos**
		8. **audio**
		9. **comments**: HTML comments found in the source code
